# saumap

基于 Android 平台的校园导航软件

## feature

1. 百度地图作为地图
2. 从当前位置的路线规划
3. 在地图上添加标注
4. 查看标注介绍
5. 删除标注
6. 语音介绍标注
7. 百米内自动介绍
